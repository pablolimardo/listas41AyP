PILAS (LIFO)

"Last In, First Out". El último que entra es el primero que sale.

  • → Uso: Historial de errores, estados previos de compuertas.
  • → Código: lista.append() y lista.pop().

COLAS (FIFO)

"First In, First Out". El primero que llega es el primero que se atiende.

  • → Uso: Sistema de alertas por nivel alto, turnos de riego.
  • → Código: lista.append() y lista.pop(0).
